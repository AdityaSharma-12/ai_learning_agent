# 🎓 AI Tutor Pro
A modern, responsive AI-powered study companion built with Python (Flask) and OpenRouter API.

## 🚀 Live Demo
Check out the live application here: https://ai-learning-agent-1.onrender.com

## ✨ Features
* Real-time AI Chat: Get instant answers to complex academic questions.
* Modern UI: Clean, bubble-chat interface optimized for mobile and desktop.
* Free Tier Integration: Utilizes OpenRouter's free models for accessible learning.
* Responsive Design: Built with CSS Flexbox for a seamless experience on all devices.

## 🛠️ Tech Stack
* Frontend: HTML5, CSS3, JavaScript (Fetch API)
* Backend: Python, Flask
* API: OpenRouter (accessing Llama 3 / Gemma models)
* Deployment: Render
* Production Server: Gunicorn
